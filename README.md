# Nimbus Adaptive Learning Platform

## Overview
A MERN-based adaptive learning system that dynamically assembles quizzes based on learner performance.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- JWT-based authentication
- Adaptive question selection
- Instructor analytics
- Dynamic quiz modes (Diagnostic, Formative, Summative)

## Run locally
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm start
